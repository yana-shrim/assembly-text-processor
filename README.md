# Assembly Text Processing System

A low-level text processing application built using x86 Assembly language.

## Features
- Counts number of characters (excluding spaces)
- Counts number of words
- Detects the most frequent character
- Replaces characters dynamically
- Supports keyboard input and screen interaction

## Technologies
- x86 Assembly
- DOS Interrupts (INT 21h, INT 16h)

## Description
This project demonstrates low-level programming skills, including memory handling, string processing, and direct interaction with system interrupts.
This project was built from scratch without high-level libraries, focusing on performance and direct system interaction.
